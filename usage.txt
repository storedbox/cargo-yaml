Generate a Cargo.toml manifest from an YAML template

USAGE:
    cargo yaml [<options>] [--] [<cargo subcommand> [<args>...]]
    cargo yaml [<options>] [-- [<cargo options>]]

FLAGS:
    -h --help                   Print this message
    -o --manifest-path <PATH>   Output path for the generated manifest
    -i --template-path <PATH>   Input path for the template file
    -v --verbose                Use verbose output
    -q --quiet                  No output printed to stdout
    -c --color <WHEN>           Coloring: auto, always, never
